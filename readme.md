# 概述

Developmental Research 是一个学术研究辅助技能，帮助用户进行教学文档的编写和研究工作。

# 命令

## 命令要点说明
- 带有括号的参数表示可缺省。
- 当括号内的参数为"文件"时，用户如选择缺省，则使用最新的文档文件
- 当括号内的参数为"文段"时，用户如选择缺省，则根据用户的需求，先寻找对应的段落，再进行细化讲解

## 命令集

| 命令 | 形式 | 说明 |
|------|------|------|
| 初级研究 | `/yanjiu c "主题" "文件"` | 编辑指定文件，创建初级研究内容，包含知识讲解和思维导图 |
| 文本细化 | `/yanjiu x "需求" （"文段"）` | 对文档中指定的段落进行细化讲解 |
| 问题解答 | `/yanjiu q "问题" （"文件"）` | 回答用户问题，并在文档对应位置添加注释 |
| 补充级研究 | `/yanjiu b "关键词" （"文件"）` | 在原文档中补充完善指定关键词的细化讲解 |
| 全局细化 | `/yanjiu g （"文件"）` | 对文档中每一章节内容的细化工作进行全局细化工作 |
| 文档整理 | `/yanjiu r （"文件"）` | 清理垃圾内容、整理文档架构、优化文档表达 |
